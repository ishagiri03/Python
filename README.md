<h2 align="center">🚀 How to Run the Projects</h2>

<p align="center">
  Each project in this repository is designed to be <b>simple to execute</b> and
  <b>independent of external frameworks</b>, allowing easy testing and experimentation.
</p>

<br/>

<h3 align="center">📥 Step 1: Clone the Repository</h3>

<p align="center">
  Clone the repository to your local machine using Git:
</p>

<p align="center">
  <code>git clone https://github.com/gavneet0030/Python_projects.git</code>
</p>

<br/>

<h3 align="center">📂 Step 2: Navigate to a Project Directory</h3>

<p align="center">
  Move into the specific project folder you want to run:
</p>

<p align="center">
  <code>cd Python_projects/&lt;project_folder_name&gt;</code>
</p>

<p align="center">
  <i>Example:</i><br/>
  <code>cd Python_projects/number_guess</code>
</p>

<br/>

<h3 align="center">▶️ Step 3: Run the Python Script</h3>

<p align="center">
  Execute the Python file using:
</p>

<p align="center">
  <code>python main.py</code>
</p>

<p align="center">
  <i>
    If a project uses a different filename, refer to the files inside that project folder.
  </i>
</p>

<br/>
